基于 [Rum](https://github.com/rumsystem/quorum) 实现的社交产品

live 版本：https://feed.base.one

## 启动前端服务

```
yarn install
yarn dev
```

## 启动后端服务

```
cd server
yarn install
yarn dev
```

## 访问服务

http://localhost:3000
